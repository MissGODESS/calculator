# Professional Calculator

A modern professional calculator built using HTML, CSS, and JavaScript.

## Features
- Basic arithmetic (+, -, *, /, %)
- Clear (AC) button
- Delete (DEL) button
- Decimal support
- Responsive design
- Modern UI

## How to Use
1. Open index.html in your browser
2. Click numbers and operators
3. Press = to calculate
4. AC to clear everything
5. DEL to remove last digit
